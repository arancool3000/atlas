This is the Ember assistant. Controlling your computer (desktop Ember's job) isn't possible inside an iPad app — use the Mac/PC app or Ember Link for that.

🛡 File safety

Pick files (from the Files app, Downloads, iCloud Drive…) and Ember checks them locally for danger signs — executable types, macro documents, double-extension tricks and the EICAR test virus — and shows each file's SHA-256. Nothing is uploaded; it all runs on this device.

Heuristic safety check — a helpful first pass, not a replacement for full antivirus. Tip: paste a file's SHA-256 into VirusTotal for a deeper check.
