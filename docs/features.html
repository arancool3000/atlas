Features

Built to actually do the work.

Ember isn't a chatbot with a search box bolted on — it operates your real desktop, end to end.

Vision & control

It looks at the screen the way you do, then acts precisely.

Accurate screen reading

On-device OCR (Apple Vision / Windows OCR) reads every visible label and returns its exact clickable center. HiDPI/Retina scaling is measured at runtime so clicks always land.

Smart click, no guessing

Resolves targets via the accessibility tree first, OCR second — and refuses low-confidence matches instead of misclicking.

Full input control

Mouse, keyboard, drag, scroll, type, paste, key-combos, window management — everything you'd do by hand.

Intelligence

Free to run, with a flagship brain on tap for the hard parts.

Gemini, free

Runs day-to-day on Google's free tier — fast, capable, no bill.

Claude Opus 4.8

Switch to Claude with adaptive thinking + streaming and prompt caching for deep reasoning and long agentic tasks.

Self-correcting loop

Plans, acts, checks the screen, and recovers — with a step budget and graceful stop reasons so it never spins silently.

Reach & integrations

Real browser, via DOM

Drives Chrome/Edge over the DevTools protocol — reads pages, fills forms, clicks by text or selector, handles tabs.

Files & shell

Find, organize, rename, zip, search inside files, run shell commands, diff, and inspect — with confirmation on anything destructive.

System diagnostics

Reads logs, processes, performance, drivers, and more to actually debug your machine.

Ember Link (phone)

Mirror your screen to your phone over Wi-Fi with a trackpad, keyboard, and a chat tab. PIN-protected, brute-force throttled.

Voice chat

Continuous listen → act → speak turns for true hands-free operation.

Automations & memory

Window-triggered rules, scheduled tasks, and persistent memory of facts about your system across sessions.

Trust & upkeep

Safety gating

Destructive shell commands, data exfiltration (email/HTTP POST), and disruptive key-combos require explicit confirmation.

Background auto-update

Checks for new releases on launch and updates itself in place — verified by checksum, with a rollback backup.

macOS & Windows

One agent, both platforms, same capabilities — including the phone remote and auto-update.
